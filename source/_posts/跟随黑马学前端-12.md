- #### 案例一：产品模块

  ![product](./跟随黑马学前端-12.assets/product.png)

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>综合案例-产品模块</title>
    <style>
        * {
            margin: 0px;
            padding: 0px;
        }

        body {
            background-color: #f5f5f5;
        }

        a{
            color: #333;
            text-decoration: none;
        }

        .box {
            width: 298px;
            height: 451px;
            background-color: white;
            /* 让块级的盒子水平居中对齐 */
            margin: 100px auto;
        }

        .box img {
            /* 图片宽度和盒子一样宽 */
            width: 100%;
        }

        .review {
            height: 70px;
            font-size: 14px;
            /* 因为这个段落没有width属性，所以padding不会撑开盒子的宽度 */
            padding: 0 28px;
            margin-top: 30px;
        }

        .appraise {
            font-size: 12px;
            color: #b0b0b0;
            margin-top: 20px;
            padding: 0 28px;
        }

        .info h4 {
            display: inline-block;
            font-weight: 400;
        }

        .info span {
            color: #ff6700;
        }

        .info em{
            font-style: normal;;
            color : #ebe4e0;
            margin: 0 6px 0 15px;
        }

        .info {
            font-size: 14px;
            margin-top: 15px;
            padding: 0 28px;
        }
    </style>
</head>

<body>
    <div class="box">

        </div>
    </div>
</body>

</html>
```





#### 总结

---

布局为什么用不同的盒子

* 标签都是有语义的，合理的地方用合理的标签。比如产品标题就用h，大量文字就用p

为啥用这么多类名

* 类名就是给每个盒子起了一个名字，可以更好地找到这个盒子，选取盒子更加容易，后期维护也方便

到底用margin还是用padding

* 大部分情况下两者都可以混用。两者各有优缺点，但是要根据实际情况，总是有更简单的方法实现。
* 上下用margin左右用padding

自己做没有思路

* 布局有很多种实现方式，可以先模仿老师的写法，然后再做出自己的风格

      /* 也可以加到*里面 也是有效的 */
      list-style: none;

    .box {
      width: 248px;
      height: 163px;
      border: 1px solid #ccc;

      margin: 100px auto;
    }

    .box h3 {
      height: 32px;
      border-bottom: 1px dotted #ccc;
      font-size: 14px;
      font-weight: 400;
      /* 行高等于盒子的高度，就可以做到垂直居中 */
      line-height: 32px;
      padding-left: 15px;
    }

    .box ul li a {
      font-size: 12px;
      color: #666;
      text-decoration: none;
    }

    .box ul li a:hover {
      text-decoration: underline;
    }

    .box ul li{
      height: 23px;

    }

    }

</head>
